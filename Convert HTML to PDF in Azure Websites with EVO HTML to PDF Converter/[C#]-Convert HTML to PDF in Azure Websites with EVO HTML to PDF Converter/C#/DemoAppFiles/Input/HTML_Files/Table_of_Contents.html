Part 1 of a Table of Contents Defined in HTML

Any HTML element can be marked to become an item in the table of contents in the generated PDF document using the data-toc-item="true" attribute. Additionally you can define the table of contents item level using the data-toc-level attribute and the table of contents item title using the data-toc-title attribute.
